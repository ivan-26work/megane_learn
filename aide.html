Guide d'utilisation

Simulateurs

Utilisez les sliders pour modifier U, R, L, C, f. Les résultats s'actualisent en temps réel. Le diagramme de Fresnel s'adapte automatiquement.

Cours

Accédez aux cours détaillés pour chaque circuit. Chaque variable est expliquée avec un exemple concret.

Favoris

Appui long (700ms) sur le nom d'un circuit pour l'ajouter aux favoris. Cliquez sur ❤️ pour ajouter/retirer instantanément.

Convertisseur

20 types de conversion : tension, courant, résistance, capacité, fréquence, etc. Choisissez l'unité d'entrée et de sortie.

Raccourcis clavier

  • Esc - Fermer la sidebar
  • Ctrl + K - Focus sur la recherche (index)
  • Ctrl + H - Retour à l'accueil

Installation PWA

  • Android / Chrome : Menu ⋮ → "Installer l'application"
  • iOS / Safari : Partager → "Ajouter à l'écran d'accueil"
  • Windows / Edge : Menu ⋮ → "Applications" → "Installer ce site"
